Artikel 1 Definitie van registratieobject, entiteiten en attributen

Registratieobject

Naam Overheidsbesluit bodemverontreiniging
Code SLD
Definitie

Het geheel van gegevens dat betrekking heeft op de status van een bepaalde locatie in het kader van milieuhygiënische bodemkwaliteit.
Populatie

De populatie bodemlocaties in de basisregistratie ondergrond omvat de locaties die een aanduiding hebben op basis van de Wet bodembescherming (Wbb), en de locaties van gemeenten die geen bevoegd gezag Wbb zijn, maar waar verdachte activiteiten zijn geregistreerd en/of de status voldoende onderzocht aan is toegekend op basis van een milieuhygiënisch bodemonderzoek.
Ieder object heeft ter identificatie een eigen BRO-ID.
De huidige gegevensdefinitie beperkt zich tot het kwaliteitsregime IMBRO/A.
De huidige werkversie beperkt zich tot de Wet Bodembescherming (Wbb).

Bodemlocatie

Type gegeven Entiteit
Definitie

De eenheid of verzameling van eenheden met een vermoeden of bewezen bodemverontreiniging die als één geheel wordt aangepakt bij onderzoek en sanering.
Toelichting

De grens van een locatie valt niet per se samen met de grens van een geval van bodemverontreiniging. Eén locatie kan meerdere gevallen bevatten. Een geval van bodemverontreiniging kan ook de grens van de bodemlocatie overschrijden.
Bodemlocaties kunnen elkaar overlappen.

locatiecode bevoegd gezag

Type gegeven Attribuut van Bodemlocatie
Definitie

De code die door het bevoegd gezag Wet bodembescherming wordt gebruikt om de bodemlocatie te identificeren.
Juridische status Authentiek
Kardinaliteit 0..1
Domein
  Naam Tekst 11
Regels IMBRO/A

De code moet uniek zijn binnen de set bodemlocaties van het registatieobjecttype Overheidsbesluit bodemverontreiniging (SLD).
Materiële geschiedenis Nee
Toelichting

De code is als volgt opgebouwd:

  • afkorting bevoegd gezag - 2 letters
  • geografische aanduiding (gemeente code) - 4 cijfers
  • uniek volgnummer binnen het beheersgebied - 5 cijfers

Wanneer de bodemlocatie een gemeentelijke locatie betreft van een gemeente die geen bevoegd gezag Wbb is, kan het gegeven ontbreken.

Voor historische gegevens onder IMBRO/A is de status niet altijd vastgelegd, en kan het gegeven ontbreken.

Toelichting

De locatiecode BIS wordt door of voor de bronhouder gebruikt om het object in het BIS te kunnen vinden.
De code wordt aangemaakt op het moment dat een locatie voor het eerst in een bodem informatie systeem (BIS) wordt geregistreerd. De code is als volgt opgebouwd:

  • code BIS-systeem - 2 letters
  • geografische aanduiding (gemeente code) - 4 cijfers
  • uniek volgnummer binnen het beheersgebied - 5 cijfers.

De BIS locatiecode is niet altijd uniek binnen de set bodemlocaties van het registatieobjecttype Overheidsbesluit bodemverontreiniging (SLD).

status Wbb

Type gegeven Attribuut van Bodemlocatie
Definitie

De fase waarin de bodemlocatie zich bevindt binnen het kader van de Wet bodembescherming (Wbb).
Juridische status Authentiek
Kardinaliteit 0..1
Domein
  Naam VervolgactieWbb
  Type Waardelijst uitbreidbaar
Materiële geschiedenis Ja
Toelichting

De bodemlocatie doorloopt een aantal stadia. De stadia zijn gekoppeld aan de Wet bodembescherming (Wbb). De status geeft aan in welk stadium de locatie zich bevindt. De status wordt bepaald door het bevoegd gezag Wbb op basis van een besluit of op basis van de onderzoeksresultaten uit het milieuhygiënisch bodemonderzoek. Het kan voorkomen dat een niet bevoegd gezag Wbb de status 'voldoende onderzocht' (ID = 20) heeft toegekend aan de bodemlocatie op basis van een milieuhygiënisch bodemonderzoek.
Het gegeven kan een nieuwe waarde krijgen bij een verandering. In dat geval blijft de oude waarde behouden.
Voor historische gegevens onder IMBRO/A is de status niet altijd vastgelegd, en kan het gegeven ontbreken.

Uitgevoerde sanering

Type gegeven Entiteit
Definitie

De uitgevoerde sanering bodemverontreiniging.

Regels IMBRO/A

De entiteit mag niet aanwezig zijn wanneer de waarde van het attribuut type besluit van de entiteit Besluit niet gelijk is aan 15 of 84.

Toelichting

De sanering is uitgevoerd en het bevoegd gezag heeft ingestemd met de uitgevoerde sanering, middels een beschikking tot instemming. Dat is een besluit 'instemmen uitgevoerde sanering' (besluittype ID = 15) of 'instemmen uitgevoerde sanering nieuw geval' (besluittype ID = 84). Wanneer het bevoegd gezag Wbb niet heeft ingestemd met de uitgevoerde sanering 'niet instemmen uitgevoerde sanering' (besluittype ID = 16) is de saneringscontour niet vastgelegd.

Toelichting

Wanneer de registerbeheerder een registratieobject uit registratie heeft genomen, zijn de gegevens niet langer beschikbaar voor andere afnemers dan bronhouder en dataleverancier. De registerbeheerder zal een registratieobject alleen bij hoge uitzondering uit registratie nemen en alleen na akkoord van de bronhouder. Aan de beslissing gaat een proces van zorgvuldige afweging vooraf en dat komt tot uitdrukking in de regel dat een registratieobject slechts een keer uit registratie kan worden genomen.
