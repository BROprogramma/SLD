Gegevensdefinitie

Registratieobject

Naam Overheidsbesluit bodemverontreiniging
Code SLD
Definitie

Het geheel van gegevens dat betrekking heeft op de status van een bepaalde locatie in het kader van milieuhygiënische bodemkwaliteit.
Populatie

De populatie bodemlocaties in de basisregistratie ondergrond omvat de locaties die een aanduiding hebben op basis van de Wet bodembescherming (Wbb), en de gemeentelijke locaties waar verdachte activiteiten zijn geregistreerd en/of de status voldoende onderzocht aan is toegekend op basis van een milieuhygiënisch bodemonderzoek.
Ieder object heeft ter identificatie een eigen BRO-ID.
De huidige gegevensdefinitie beperkt zich tot het kwaliteitsregime IMBRO/A.

Bodemlocatie

Type gegeven Entiteit
Definitie

De gegevens die de status van de locatie beschrijven.
Toelichting

Een eenheid of aggregatie van kleinere eenheden met een vermoede of bewezen bodemverontreiniging die als één geheel wordt aangepakt bij onderzoek en sanering. De grens van een locatie valt niet per se samen met de grens van een geval van bodemverontreiniging. Eén locatie kan soms verschillende gevallen bevatten (of andersom).
Bodemlocaties kunnen overlappen.

dossier-ID bodem informatiesysteem

Type gegeven Attribuut van Bodemlocatie
Definitie

Juridische status Authentiek
Kardinaliteit 1
Domein
  Naam Tekst 11
Materiële geschiedenis Nee
Toelichting

De code is uniek binnen de set bodemlocaties van het registatieobjecttype Overheidsbesluit bodemverontreiniging (SLD).
De code wordt aangemaakt als een locatie voor het eerst in een BIS wordt geregistreerd. De code is als volgt opgebouwd:

  • code BIS-systeem - 2 letters
  • geografische aanduiding (gemeente code) - 4 cijfers
  • uniek volgnummer binnen het beheersgebied - 5 cijfers

vervolgactie Wbb

Type gegeven Attribuut van Bodemlocatie
Definitie

De actuele fase waarin de bodemlocatie zich bevindt binnen het kader van de Wet bodembescherming (Wbb).
Juridische status Authentiek
Kardinaliteit 0..1
Domein
  Naam VervolgactieWbb
  Type Waardelijst uitbreidbaar
Materiële geschiedenis Ja
Toelichting

De bodemlocatie doorloopt een aantal stadia. De stadia zijn gekoppeld aan de Wet bodembescherming (Wbb). De vervolgactie geeft aan in welk stadium de locatie zich bevindt. De vervolgactie wordt bepaald door het bevoegd gezag Wbb op basis van een besluit of op basis van de onderzoeksresultaten uit het milieuhygiënisch bodemonderzoek. Het kan voorkomen dat een niet bevoegd gezag de status 'voldoende onderzocht' (VervolgactieWbb ID = 20) heeft toegekend aan de bodemlocatie op basis van een milieuhygiënisch bodemonderzoek.
Onder IMBRO/A is de vervolgactie niet altijd vastgelegd, en kan het gegeven ontbreken.

Uitgevoerde sanering

Type gegeven Entiteit
Definitie

De uitgevoerde sanering bodemverontreiniging.

Regels IMBRO/A

De entiteit mag niet aanwezig zijn wanneer de waarde van het attribuut type besluit van de entiteit Besluit niet gelijk is aan 15 of 84.

Toelichting

Saneren is het verwijderen van de verontreiniging of het beperken van de risico's.
De sanering is uitgevoerd en het bevoegd gezag heeft ingestemd met de uitgevoerde sanering, middels een beschikking tot instemming. Dat is een besluit 'instemmen uitgevoerde sanering' (besluittype ID = 15) of 'instemmen uitgevoerde sanering nieuw geval' (besluittype ID = 84). Wanneer het bevoegd gezag Wbb niet heeft ingestemd met de uitgevoerde sanering 'niet instemmen uitgevoerde sanering' (besluittype ID = 16) is de saneringscontour niet vastgelegd.
